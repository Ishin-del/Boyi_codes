# MLs





## 线性模型系列

|      模型      |             优点             |            缺点            |           最佳适用场景           |
| :------------: | :--------------------------: | :------------------------: | :------------------------------: |
|    线性回归    |      简单易解释，计算快      |  无法处理非线性，易过拟合  | 特征与目标线性关系明确的回归问题 |
|     岭回归     |    解决共线性，防止过拟合    |        不做特征选择        |    特征多且可能相关的回归问题    |
|   Lasso回归    |    自动特征选择，模型简洁    |   特征多且相关时可能选偏   |    高维稀疏数据（如基因数据）    |
|    弹性网络    |   结合L1和L2优势，鲁棒性强   |       需要调两个参数       |    特征数量>样本量的回归问题     |
|    逻辑回归    |     输出概率，可解释性强     |    只能处理线性可分数据    |   二分类问题（如垃圾邮件识别）   |
| 多分类逻辑回归 |  扩展简单，适合多类平衡数据  |       类别多时有偏差       |     类别少且线性可分的多分类     |
|   SGD分类器    |  训练速度快，适合大规模数据  | 对学习率敏感，需数据标准化 |     在线学习、大规模分类任务     |
|      LDA       |   降维+分类一体，抗噪声强    | 假设数据正态分布，线性边界 |   高维数据分类（如图像、文本）   |
|      QDA       | 捕捉非线性关系，分类边界灵活 |   需要更多样本，易过拟合   |  样本量大且类别分布差异大的数据  |
